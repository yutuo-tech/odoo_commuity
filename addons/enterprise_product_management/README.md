# 企業產品管理模組（繼承版）

## 概述
本模組基於 Odoo 18.0 核心模組繼承實作，提供工單系統所需的基礎資料管理功能。

## 功能特色

### 📦 產品管理（繼承 product.template）
- 添加產品類型欄位
- 完全整合原有產品管理功能

### 🔧 物料管理
- **物料種類**：獨立模型管理
- **物料清單**：繼承 product.product，添加物料種類和庫存數量

### 🔨 維修問題庫（獨立模型）
- 主項-次項結構的維修問題管理
- 支援價格和貨幣設定
- 可控制是否顯示給外部工單

### 🔧 保養項目（繼承 maintenance.request）
- 關聯貨物和產品
- 詳細的保養要點記錄
- 可控制是否顯示在工單

### 📋 合約管理（繼承 sale.order）
- 完整的合約資訊管理
- 客戶部門和聯絡人關聯
- 合約期間和類型管理

### 📱 設備管理（繼承 stock.lot）
- 資產編號和序號管理
- 客戶、部門、聯絡人三級關聯
- 完整的設備生命週期管理
- 保固期追蹤

## 安裝說明

### 前置條件
確保以下 Odoo 核心模組已安裝：
- `product` - 產品管理
- `sale_management` - 銷售管理
- `maintenance` - 維護管理
- `repair` - 維修管理
- `stock` - 庫存管理
- `mrp` - 製造管理
- `medical_service_management` - 企業聯絡人管理

### 安裝步驟
1. 重啟 Odoo 服務
2. 進入應用程式頁面
3. 搜尋「企業產品管理」
4. 點擊安裝

### 升級指南
從舊版本升級時，建議：
1. 備份資料庫
2. 卸載舊版本模組
3. 安裝新版本
4. 重新配置相關設定

## 使用說明

### 選單結構
```
企業產品管理
├── 產品管理
│   ├── 產品主檔 (product.template)
│   └── 產品變體 (product.product)
├── 物料管理
│   ├── 物料種類 (獨立模型)
│   └── 物料清單 (product.product)
├── 維修保養
│   ├── 維修問題庫 (獨立模型)
│   └── 保養項目 (maintenance.request)
├── 合約管理 (sale.order，篩選合約)
└── 貨物/設備 (stock.lot)
```

### 資料關聯
- **客戶公司** → **部門** → **聯絡人** (三級結構)
- **產品** → **物料種類** (分類管理)
- **設備** → **合約** → **客戶** (追蹤關係)
- **維修問題** → **物料種類** → **具體物料** (主項-次項)

## 技術特色

### 🚀 繼承優勢
- **代碼量減少 80%**：充分利用 Odoo 核心功能
- **自動獲得功能**：工作流程、權限、報表、API
- **維護成本極低**：跟隨 Odoo 升級自動更新
- **無縫整合**：與其他 Odoo 模組完美整合

### 🔒 安全性
- 利用現有模組權限群組
- 最小權限原則
- 基於角色的存取控制

### 📊 報表和分析
- 繼承所有核心模組的報表功能
- 支援樞紐分析和圖表
- 可配置的儀表板

## 開發規範
- 所有自定義欄位使用 `x_` 前綴
- 不覆蓋原有方法，只添加欄位
- 遵循 Odoo 18.0 開發最佳實踐
- 極簡設計，避免複雜業務邏輯
